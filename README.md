# GLOOBA

This is a web application for GLOOBA, a social media platform.

## Frontend

The frontend is built with plain HTML, CSS, and JavaScript.

### Running the frontend

To run the frontend locally, you can use a simple Python HTTP server.

1.  Navigate to the root directory of the project.
2.  Run the following command:
    ```bash
    python -m http.server 8080
    ```
3.  Open your web browser and go to `http://localhost:8080`.

## Backend

The backend is a Flask application located in the `backend/` directory.
